# Ramruaycha (ร่ำรวยชา POS)

Welcome to the Ramruaycha project! 

## Overview
This is a web application project for a Point of Sale (POS) system for the "Ramruaycha" tea shop.

## Technologies Used
- HTML/CSS/JavaScript
- Firebase

---

## 📖 คู่มือการใช้งานระบบ POS "ร่ำรวยชา"

### 1. การเข้าสู่ระบบ (Login)
* เมื่อเปิดหน้าระบบขึ้นมา ระบบจะถามรหัสผ่าน 
* **รหัสผ่านเข้าใช้งานคือ:** `191512` 

### 2. เมนูการใช้งานหลัก (แถบเมนูด้านล่าง)
ระบบถูกแบ่งออกเป็น 5 หน้าหลัก ดังนี้:

**🛒 1. ขาย (POS) - สำหรับรับออเดอร์หน้าร้าน**
* **การสั่งเครื่องดื่ม:** เลือกหมวดหมู่ (กาแฟ, ชา, นม, โซดา) และกดเลือกเมนูที่ลูกค้าสั่ง
* **การเพิ่มท็อปปิ้ง:** ให้กดเลือกเมนูเครื่องดื่มก่อน แล้วค่อยกดเลือกท็อปปิ้ง (เช่น ไข่มุก, วิปครีม) ท็อปปิ้งจะถูกบวกเข้าไปในรายการล่าสุด
* **การชำระเงิน:**
   * **💵 เงินสด:** เมื่อกดปุ่มเงินสด ระบบจะให้กรอกจำนวนเงินที่รับมาเพื่อคำนวณเงินทอน (มีปุ่มลัดให้กดเร็วๆ เช่น รับมาพอดี, 100, 500, 1000)
   * **📱 เงินโอน:** ระบบจะสร้าง QR Code พร้อมเพย์ให้อัตโนมัติ (ผูกกับเบอร์ `0964032659`) ตามยอดชำระ ให้ลูกค้าสแกนจ่าย เมื่อเงินเข้าแล้วให้กด "เงินเข้าแล้ว"

**📉 2. ต้นทุน (Expenses) - สำหรับบันทึกรายจ่าย**
* ใช้สำหรับบันทึกต้นทุนรายวัน เช่น ซื้อแก้ว, น้ำแข็ง, วัตถุดิบ
* กด "+ เพิ่มรายการ" เพื่อเพิ่มบรรทัด ใส่ชื่อรายการและจำนวนเงิน จากนั้นกดปุ่ม "บันทึกต้นทุนทั้งหมด"

**📊 3. สรุป (Dashboard) - สำหรับดูยอดขายและกำไร**
* ระบบจะสรุป **ยอดขาย, ต้นทุน, และกำไรสุทธิ** ให้ดูอย่างชัดเจน
* สามารถสลับดูได้ทั้งแบบ รายวัน, รายเดือน, และรายปี (พร้อมกราฟแท่ง)
* หากยอดแสดงผลไม่ตรงหรือไม่โหลด สามารถกดปุ่ม **"ซ่อมยอด" (รูปประแจ)** เพื่อให้ระบบคำนวณยอดของวันนั้นใหม่ได้

**🕒 4. ประวัติ (History) - สำหรับดูบิลย้อนหลัง**
* สามารถเลือกวันที่เพื่อดูประวัติ "รายการขาย" และ "รายการต้นทุน" ได้
* หากคิดเงินผิด หรือบันทึกต้นทุนผิด สามารถกดที่ **รูปถังขยะ** หลังรายการเพื่อลบทิ้งและยกเลิกยอดนั้นๆ ได้

**⚙️ 5. ตั้งค่า (Settings) - สำหรับจัดการเมนู**
* ใช้สำหรับ เพิ่ม/แก้ไข/ลบ เมนูเครื่องดื่ม และ ท็อปปิ้ง ต่างๆ ในร้าน
* สามารถแก้ไขชื่อ และตั้งราคาใหม่ได้ เมื่อทำเสร็จอย่าลืมกด "บันทึกการตั้งค่า"
